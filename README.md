# analyze-to-slot
【使用技術】
・フロント
html,css,jquery
・バックエンド
pyrhon、django
・インフラ
heroku

【概要】
別アプリケーションとして作成した「slot_scraping」でDBに保存しているデータの参照機能。
ある店舗のハナハナシリーズ(ハナハナ鳳凰、ツインドラゴン、プレミアム)の3機種のデータを参照する。

【URL】
https://analyze-to-slot.herokuapp.com/slot/

heoku環境で別アプリとして取得しているデータの参照する。
